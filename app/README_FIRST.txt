============================================================
           INSTALLATION INSTRUCTIONS (LINUX)
============================================================

To install "Vault" on your Linux system, follow these 
simple steps:

1. Open a terminal in this folder.
   (You can right-click on an empty space in the folder 
   and select "Open in Terminal").

2. Copy and paste the following command to give 
   execution permissions to the installer:

   chmod +x install_linux.sh

3. Run the installer with this command:

   ./install_linux.sh

4. Follow the on-screen instructions (select your language).
   You will be asked for your administrator password to finish.

5. Once the installation is finished, you can open "Vault"
   from your system's application launcher (press the 
   "Super/Windows" key and type "Vault"). 
   
   You can also open it by typing 'vault' in the terminal.

Note: Inside the application (Settings), you can change 
the language to any of the 13 available options 
(Spanish, Chinese, Arabic, French, etc.).
